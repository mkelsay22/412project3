Detailed Description

Main driver program for the Load Balancer Simulation.

Author
Your Name
Date
2024
Version
1.0

This program simulates a load balancer system with multiple web servers. It allows users to configure the number of servers, simulation time, and generates a full queue of requests. The system dynamically allocates and deallocates servers based on load conditions.
